# DC101_Game_AngelesJheanCassey
This game is an interactive project created for the DC101 course. It challenges players to complete levels by matching items, solving simple tasks, and improving their memory and focus. The game features a clean interface and easy-to-use controls.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Game
1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` to start the game.
