# Memory Match Game

## Description
This game is a simple and interactive memory match game created for the DC101 course.
The goal of the game is to match all card pairs by flipping them over.
It helps improve memory, focus, and attention.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Game
1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` to start the game.
